## üìñ Overview

The **22_banu-airport-management-fullstack** is a complete solution for managing airports. This system allows you to handle flights, passengers, and bookings effortlessly. Built with Django REST Framework for the backend and React for the frontend, it offers a modern dashboard for a smooth user experience.

## ‚ú® Features

- **User-Friendly Dashboard**: Navigate easily through flights, bookings, and passenger management.
- **Flight Management**: Add, update, and delete flights with a few clicks.
- **Passenger Management**: Keep track of passengers and their bookings.
- **Secure Authentication**: Uses JWT for secure user login.
- **Responsive Design**: Works on both desktop and mobile devices.
